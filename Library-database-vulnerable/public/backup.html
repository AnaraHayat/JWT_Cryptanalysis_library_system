Backup & Restore Database

Backup Database

Create a backup of the entire library database. Backups are saved to the specified location on the server.

Information

Backups include all tables: Books, Members, Authors, Publishers, Borrowing, Fines, Users, and more.

Restore Database

Restore the library database from a previously created backup file.

Warning

Restoring will overwrite all current data in the database. This action cannot be undone!
